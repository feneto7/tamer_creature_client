# 🎮 PIXEL_TAMERS - Client

Este é o repositório do **Frontend (Client)** do Pixel Tamers, construído com React, Vite e TypeScript.
O jogo é um RPG de criaturas colecionáveis pixel art, desenvolvido com foco mobile-first.

---

## 🚀 Como Rodar o Cliente Localmente

### 1. Pré-requisitos
- **Node.js** (versão 18 ou superior recomendada)

### 2. Instalação das Dependências

Execute o comando abaixo na raiz deste repositório para instalar as dependências:

```bash
npm install
```

### 3. Executando o Projeto

Inicie o servidor de desenvolvimento do Vite:

```bash
npm run dev
```

O frontend geralmente rodará em `http://localhost:5173`. 
**Importante:** Certifique-se de que o repositório do servidor (backend) também esteja configurado e rodando para que o jogo funcione corretamente e consiga consumir as APIs.
